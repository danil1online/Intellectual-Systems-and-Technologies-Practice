    * –°—Ä–µ–¥–Ω—è—è –∫–≤–∞–¥—Ä–∞—Ç–∏—á–µ—Å–∫–∞—è –æ—à–∏–±–∫–∞ (MSE): —Å—Ä–µ–¥–Ω—è—è –∫–≤–∞–¥—Ä–∞—Ç–∏—á–µ—Å–∫–∞—è –æ—à–∏–±–∫–∞ (MSE) - —ç—Ç–æ —Å—Ä–µ–¥–Ω–µ–µ –∑–Ω–∞—á–µ–Ω–∏–µ –∫–≤–∞–¥—Ä–∞—Ç–∞ –æ—à–∏–±–∫–∏. –û–Ω–∞ –±–æ–ª–µ–µ –ø–æ–ø—É–ª—è—Ä–Ω–∞, —á–µ–º —Å—Ä–µ–¥–Ω—è—è –∞–±—Å–æ–ª—é—Ç–Ω–∞—è –æ—à–∏–±–∫–∞, –ø–æ—Ç–æ–º—É —á—Ç–æ –±–æ–ª—å—à–µ –æ—Ä–∏–µ–Ω—Ç–∏—Ä–æ–≤–∞–Ω–∞ –Ω–∞ –±–æ–ª—å—à–∏–µ –æ—à–∏–±–∫–∏. –≠—Ç–æ —Å–≤—è–∑–∞–Ω–æ —Å —Ç–µ–º, —á—Ç–æ –∫–≤–∞–¥—Ä–∞—Ç–Ω—ã–π —á–ª–µ–Ω —ç–∫—Å–ø–æ–Ω–µ–Ω—Ü–∏–∞–ª—å–Ω–æ —É–≤–µ–ª–∏—á–∏–≤–∞–µ—Ç –±–æ–ª—å—à–∏–µ –æ—à–∏–±–∫–∏ –ø–æ —Å—Ä–∞–≤–Ω–µ–Ω–∏—é —Å –º–µ–Ω—å—à–∏–º–∏.
    * –°—Ä–µ–¥–Ω–µ–∫–≤–∞–¥—Ä–∞—Ç–∏—á–µ—Å–∫–∞—è –æ—à–∏–±–∫–∞ (RMSE).
    * R-–∫–≤–∞–¥—Ä–∞—Ç –Ω–µ —è–≤–ª—è–µ—Ç—Å—è –æ—à–∏–±–∫–æ–π, –Ω–æ —è–≤–ª—è–µ—Ç—Å—è –ø–æ–ø—É–ª—è—Ä–Ω–æ–π –º–µ—Ç—Ä–∏–∫–æ–π –¥–ª—è –æ—Ü–µ–Ω–∫–∏ —Ç–æ—á–Ω–æ—Å—Ç–∏ –≤–∞—à–µ–π –º–æ–¥–µ–ª–∏. –û–Ω –ø–æ–∫–∞–∑—ã–≤–∞–µ—Ç, –Ω–∞—Å–∫–æ–ª—å–∫–æ –±–ª–∏–∑–∫–æ –¥–∞–Ω–Ω—ã–µ –∫ –ø–æ–¥–æ–±—Ä–∞–Ω–Ω–æ–π –ª–∏–Ω–∏–∏ —Ä–µ–≥—Ä–µ—Å—Å–∏–∏. –ß–µ–º –≤—ã—à–µ R-–∫–≤–∞–¥—Ä–∞—Ç, —Ç–µ–º –ª—É—á—à–µ –º–æ–¥–µ–ª—å —Å–æ–æ—Ç–≤–µ—Ç—Å—Ç–≤—É–µ—Ç –≤–∞—à–∏–º –¥–∞–Ω–Ω—ã–º. –ù–∞–∏–ª—É—á—à–∞—è –≤–æ–∑–º–æ–∂–Ω–∞—è –æ—Ü–µ–Ω–∫–∞ - 1,0, –∏ –æ–Ω–∞ –º–æ–∂–µ—Ç –±—ã—Ç—å –æ—Ç—Ä–∏—Ü–∞—Ç–µ–ª—å–Ω–æ–π (–ø–æ—Ç–æ–º—É —á—Ç–æ –º–æ–¥–µ–ª—å –º–æ–∂–µ—Ç –±—ã—Ç—å –ø—Ä–æ–∏–∑–≤–æ–ª—å–Ω–æ —Ö—É–∂–µ).
